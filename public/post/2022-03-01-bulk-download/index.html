Bulk download with R and yfR

2022-03-01

Aim: Dowload crypto data from Yahoo Finance

You can download historical financial data from Yahoo Finance directly into R and process the data without having to use Excel as an intermediate step. When you download data from Yahoo Finance, you get 1 data set in 1 csv file. If you need data for multiple time series -multiple stock returns for portfolio optimization for example- using the package yfR will automate the download and format the data for further use.

In this past we are going to download data for 4 cryptos and then estimate a simple GARCH model for each. First, install the packages yfR and rugarch.
